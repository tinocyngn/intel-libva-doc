float VABlendState::global_alpha

float VABlendState::max_luma

Maximum luma value.

Valid if has VA_BLEND_LUMA_KEY. Valid range is 0.0 to 1.0 inclusive. max_luma shall be set to a sensible value larger than min_luma.

float VABlendState::min_luma

Minimum luma value.

Valid if has VA_BLEND_LUMA_KEY. Valid range is 0.0 to 1.0 inclusive. min_luma shall be set to a sensible value lower than max_luma.
